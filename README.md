<h2>自用工具库</h2>
<ul>
    <li>password: 对密码进行加密与解密</li>
    <li>jwt: token登陆验证</li>
    <li>session: session登陆保持</li>
    <li>log: 自定义日志样式</li>
    <li>to: 转换相关</li>
    <li>sort: 排序</li>
    <li>conf: 配置文件相关</li>
    <li>resp: 回复格式相关</li>
    <li>middleware: 跨域中间件</li>
    <li>mysql: mysql初始化</li>
    <li>redis: redis初始化</li>
</ul>
